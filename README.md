# **To-Do App — Проектная работа 5 спринта**

Одностраничное приложение списка дел. Пользователь может создавать задачи, редактировать их, копировать, удалять, а также сохранять текущий список в локальное хранилище. При обновлении страницы данные не пропадают.

## **Технологии**

-   **HTML5**
-   **CSS3**
-   **JavaScript**
-   Работа с **DOM**
-   Шаблоны `<template>`
-   **LocalStorage**

## **Запуск проекта**

1. Склонируйте репозиторий:

    ```
    git clone https://github.com/morsssch/to-do-pre
    ```

2. Откройте проект в любом современном браузере.
3. Файл `index.js` содержит реализацию всей логики работы приложения.
